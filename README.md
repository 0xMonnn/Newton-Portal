### Magic Newton Daily Roll

Magic Newton Daily roll bot

Clone repository:

```
git clone https://github.com/0xMonnn/Newton-Portal.git
```

Directory Magic Newton:

```
cd Newton-Portal
```

install modules

```
npm install
```

Create folder data.txt

```
nano data.txt
```

`fill_your_cookie_session`

Create Screen

```
screen -S magicnewton
```

Run

```
node index.js
```

Exit screen:
CTRL + A + D

Done!
